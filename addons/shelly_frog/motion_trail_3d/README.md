## MotionTrail3D

An implementation of a 3D trail that follows a node.  

[trail_example.webm](https://github.com/user-attachments/assets/69e7c88f-4f29-479b-a3e1-823d5afe8001)

Features:  
- In-editor preview.
- Custom material support.
- Color over time and length via vertex colors.
- Width over time and length.
- Tiled or stretched UVs.
- Optional Catmull-Rom interpolation.
- Optional simple physics simulation on trail points.
